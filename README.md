# Foodie - an e-commerce store

### Overview

Foodie is made with React, Styled Components and deployed to Firebase hosting

### Description

This app mimic an e-commerce experience and it is interactive and responsive. It allows the user to browse the items and check their details. Add or removes items from/to the cart. This app was built to showcase my simple design and to demonstrate how I use hooks especially the useReducer hook.
